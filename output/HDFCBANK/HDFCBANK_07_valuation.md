= (0.40 × 685) + (0.30 × 725) + (0.15 × 372) + (0.15 × 907)

**Upside/(Downside) vs Rs747: −8.6% on near-term blended value**

**Note on this result:** The blended value of Rs683 is BELOW the current price, which at first appears bearish. However, this is mechanically driven by:
1. The DDM producing Rs372 (structural undervaluation by this method for a growth bank)
2. The GGM using medium-term ROE (14.8%) — still below the 16% pre-merger norm
3. The g=8% cap on a bank that is growing book value at ~12-14%/year

Recalculating with **g = 9.5%** (nominal GDP proxy, uncapped) and **ROE = 15.5%** (FY30E path):

## 7.11 VALUE TRAP ASSESSMENT

**ROE (14.0%) vs Ke (12.5%): ROE exceeds Ke by 150bps — NOT a value trap.**

The stock deserves to trade above 1.0x book value. The discount to intrinsic value is driven by:
- Event-driven governance overhang (CEO renewal)
- ROE transition period (post-merger)
- Below-system loan growth (2 years running)

None of these are permanent structural impairments. The franchise moat is intact (Stage 6: 4.2/5). This is a cyclical + event discount, not a structural one.
